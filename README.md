# closing_trade_pwa

PWA for 종가매매(Closing Price Trading):
- 시간대별 체크리스트(14:30~16:00, 익일 09:00~09:05)
- 종목 후보/진입/청산 기록(저널)
- 통계(승률/평균수익/연속손실/간단 MDD)
- 로컬 저장(IndexedDB) + export/import(JSON)
- OpenClaw Telegram 알림 연동용 "복사" 버튼( iOS 자동붙여넣기 제한 대응 )

## Run
Open `docs/index.html`.
